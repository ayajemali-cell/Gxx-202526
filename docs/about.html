Welcome to Our Project - Liste de Course

Ce projet est une application web simple de liste de course.

Technologies utilisées

HTML, CSS, JavaScript

Objectifs du Projet

Gestion Simplifiée

Permettre aux utilisateurs de créer et gérer une liste de courses interactive en temps réel.

Organisation des Données

Structurer les produits par catégories (Fruits, Viandes, Laitage) pour une meilleure visibilité.

Persistance (Étape Finale)

Sauvegarder les données dans une base de données SQLite pour un accès ultérieur.

Interface Responsive

Offrir une expérience fluide sur ordinateur et sur mobile grâce à un design moderne.

Stack

  • Python Flask — lightweight WSGI web framework
  • SQL SQL — ORM with SQLite (swappable to PostgreSQL/MySQL)
  • JS Vanilla JavaScript — async fetch API, DOM manipulation
  • HTML Jinja2 templates — server-side rendering with template inheritance
  • CSS Custom CSS — responsive layout with CSS variables
